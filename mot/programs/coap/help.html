Selecting a table/data set

Use the tabs at the top of the grey form to switch between data sets. Each user may have a different set of tabs - to select one simply click on the word as you would a normal web link. The page will then be refreshed to display the selected data set. The current selected tab appears slightly raised and the title is not underlined.

Selecting an employee

An employee is select by CID ( Commerce ID ). If the employee's CID is already known you can type the CID directly in to the field provided and press enter. To search for an employee click on the words "Select New", that appear directly under the CID field. This will bring up a window with a name field. Here you can type in the employee's name or a subset of the name. e.g. to search for Joe Smith you can type "joe", "smith" or "joe smith". After typing in the name click on the search button or press enter. The window will refresh with a list of the employees who match the name given. To select an employee click on their CID - the search window will close and the main window should refresh with that employee's record. The currently selected employee's name and supervisor is displayed to the left of the CID field.

Non Easter Inch or ex-employees

The employee search window will only list current Easter Inch employees. The CID field in the employee select form will allow any valid CID however. You can use this fact to review or update records for any Motorola employee - past or present. Note however that the system will not display the employee's name or supervisor.

Updating a record

To update a record first select the correct data set and employee. If any records for that selection exist then a table containing these records will be displayed. Click on the pen in the "Edit" column to update that record. A new window will open with an update form. Change the fields by typing in new values or by selecting a new item from the pick-lists. When you have changed the fields as required, click on the update button. The update window will close and the main window will be refreshed with the updated data.

Adding a record

To update a record first select the correct data set and employee. Click on the button labelled "Add New Record". A new window will open with an update form. Fill in the fields by typing in text values or by selecting a new item from the pick-lists. When you have completed the fields as required, click on the update button. The update window will close and the main window will be refreshed with the new record.

The HR & team leader corrective action fields

The HR & team leader fields in a corrective action record expect CIDs. You can select an individual by name however by clicking the title of the field. This will open a new employee search window. The window will contain an initial list of all HR staff and all supervisors for the respective fields. Click on the employee's CID to select.

Adding a new user

To add a new user you must have access to the admin tab. First select the admin tab and then use the pick-list at the top of the form to select "Access Control" - (this is the default). Once the page has refreshed two forms will be displayed. Use the bottom form for adding a user. First type in the user core/X500 ID. If you do not know the user's ID you can look it up by clicking on the link marked "Lookup" directly under the User ID field. This will open a window that operates almost exactly like the employee search window. Next select which tabs the user will have access to by clicking on the corresponding checkboxes. When finished click on the "Add New User" button; the window will then be refreshed to include the new user.

Updating user access

To update the access privileges for a user first select the admin tab and then use the pick-list at the top of the form to select "Access Control" - (this is the default). A list of all current users will be displayed. Only the names of current Easter Inch employees are shown. Use the checkboxes on the user's row to change which tabs they have access to. You can update more than one user at the same time. When you have selected the correct checkboes click on the "Update" button to commit the changes. The window will be refreshed with the updated privileges. If you do not select any tab for a particular user then that user will be removed from the list.

Adding a pick-list item

To add a pick-list item you must have access to the admin tab. First select the admin tab and then use the pick-list at the top of the form to select the list you wish to access. A list of all current items in the list will be displayed, followed by a form to add a new item. Simply type in the name of the new item and click on the "Add New Entry" button. The window will be refreshed to include the new item in the list. You cannot update or delete an item from a pick-list.
