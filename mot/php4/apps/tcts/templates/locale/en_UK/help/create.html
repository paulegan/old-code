TCTS Help :: Overview :: Create Transaction


1.

The transaction date defaults to the current date (today). Change this to the date that the transaction is to be processed. All dates must be in the format DD/MM/YYYY. e.g. The 2nd June 2001 would be 02/06/2001.

2.

Select the person (usually a HR administrator) that submitted the transaction to the transaction centre.

3.

Enter the number of lines (one for each employee) to be added to the form. e.g. if there are 5 employees to be processed in this transaction, enter the number "4" to add 4 additional lines. Click on "Add lines". The form will refresh with the required number of lines.

4.

Enter an identifier for an employee. This will normally be the employee's commerce ID. You may however, enter any identifier once it is a string of between 3 & 8 letters or numbers.

5.

Select the type of transaction for this employee. You may select different types for each employee.

6.

Enter the date that this transaction was approved for this employee. This will default to the current date.

7.

Enter the date that this change should become effective for this employee. This will default to the current date.

8.

Click on the add button to submit the form. When this is first done, the values in the employee fields are checked for valid CIDs. The form is re-displayed with the names of the employees or a warning of the employee is not found. Click on the submit button again to create the transactions.
