TCTS Help :: Overview :: Update Users


The number of rows in the permission matrix will change depending on how many centres you have the "manage users" privilege. If only one centre is allowed, the row title will be "Access" rather than the centre name.

Users with the "submit batch" privilege will appear in the "Submitted By" lists for the appropriate center.

1.

First select the user you wish to update permissions for. The permission matrix will change to reflect that user's permissions.

2.

Change the permissions as required by selecting/unselecting the checkboxes in the permission matrix. When a checkbox is selected this indicates that the user will be permitted to perform the operation specified by the column title for the center specified by the row title.

3.

To commit the changes made in the permission matrix for the selected user, click on the "Commit Changes" button.

4.

Click on the "Delete User" button to remove all permissions from the selected user.

5.

Click this button to open the page for adding new users.
