TCTS Help :: Overview :: Search Form


This search form enables you to search & view any transaction for which you have the privileges to view for the relevant centre. It is not necessary to complete all fields on this search form. Any field left blank will match any value in that field. If you leave all fields blank then all transactions that you have access to will be displayed.

1.

If you know the transaction ID of the transaction you wish to view, you can enter it here.

2.

All transactions created together will be assigned the same batch ID. To view all transactions of the same batch, enter the batch ID here.

3.

Select a name from this list to restrict the search to all transactions submitted by that person.

4.

Use this list to restrict the search to transactions with a specific status. All transactions which have been submitted as a batch but have not yet been integrated, will have a status of "Pending".

5.

Enter an employee identifier into the employee field to find all transactions for that employee. The identifier will usually be commerce ID.

6.

Restrict the search to a particular transaction centre. If you have access to only one centre, this field will not be displayed.

7.

To view only those transactions with a transaction date after a certain date, enter that date here. Note: the transaction date is an editable field when creating a transaction; therefor it may not be the date the transaction was actually created.

8.

Click on the Search button to display those transactions matching the data in the fields. Some queries (e.g. to view all transactions) may take a while to complete - please be patient.
