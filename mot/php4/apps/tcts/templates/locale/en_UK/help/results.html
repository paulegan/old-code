TCTS Help :: Overview :: Search Results


1.

The title will display how many results were matched by the search query & which transactions from those results are currently displayed.

2.

Use the "Prev" and "Next" buttons to move forwards or backwards through the search results.

3.

Click on a transaction ID to view the details of that transaction

4.

If the transaction(s) you were searching for are not displayed, click on the "Search Again" button and try a different search query.

5.

If the search returns a single batch of transactions with each transaction having a status of pending. The "Integrate Batch" button will be displayed. Click this button to change the status of each transaction in the batch to "Open".

6.

To reject a pending batch, first select the reject reason from the list, then click on the "Reject Batch" button.
