TCTS Help :: Overview :: Manage Lists


1.

First select the list to be managed. The page will refresh with the details of the selected list.

2.

To add a new item, type in the text for the item, then click on the "Add" button. The item should now appear highlighted in the list.

3.

To remove a list item, first select the item in the list, then click on the remove button. The text will be moved to the "New Item" input field. You can modify the entry an add it again if so desired.

4.

When you are satisfied that the list has the correct items, click on the "Commit Changes" button to update the database. Changes will be discarded if you do not do this.
